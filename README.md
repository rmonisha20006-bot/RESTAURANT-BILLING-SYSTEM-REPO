# Restaurant Billing System (Java CLI)

A simple **console-based Restaurant Billing System** written in Java.

## Features
- Menu display
- Add items to order
- Remove items from order
- GST calculation (5%)
- Itemized receipt
- Modular design using classes and methods

## Project Structure
```
restaurant-billing-system
│
├── RestaurantBillingSystem.java
└── README.md
```

## Compile
```
javac RestaurantBillingSystem.java
```

## Run
```
java RestaurantBillingSystem
```

## Example Output
```
===== RESTAURANT BILLING SYSTEM =====
1. Show Menu
2. Add Item to Order
3. Remove Item
4. Generate Bill
5. Exit
```
